· 安装教程

1.鼠标右键解压到“Minitab 16”

2.选中mtbzh1610su.exe，鼠标右击选择“以管理员身份运行”

3.点击“下一步”

4.勾选我接受此许可证协议中的条款,点击“下一步”

5.点击“下一步”

6.点击“否”

7.点击“安装”

8.软件正在安装，请耐心等待

9.点击“完成”

10.打开刚刚解压后的文件夹，复制【MtbAuth.dll】文件

11.打开路径至C:\Program Files (x86)\Common Files\Minitab Shared\Auth\SU

12.粘贴刚刚复制的文件，点击“替换目标中的文件”

13.点击“继续”

14.双击图标，打开软件

15.安装结束
