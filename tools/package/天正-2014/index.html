· 软件介绍

天正建筑CAD基于AutoCAD应用而开发，因此对软硬件环境要求取决于 AutoCAD 平台的要求，支持AutoCAD多个图形平台的安装和运行，天正对象除了对象编辑命令外，还可以用夹点拖动、特性编辑、在位编辑、动态输入多种手段调整对象参数。

· 软件下载

(特别声明: 安装包获取方式有很多，这一步非必须，大家凭喜好关注即可！)

· 安装教程

1.鼠标右键解压到“天正建筑2014”

2.选中天正建筑2014，鼠标右击选择“以管理员身份运行”

3.勾选我接受许可证协议中的条款，点击“下一步”

4.点击“下一步”

5.选择软件安装路径，点击“下一步”

6.点击“下一步”

7.软件正在安装，请耐心等待

8.点击“完成”

9.双击打开【破解文件】文件夹

10.复制这两个文件

11.选中软件图标，鼠标右击选择“打开文件位置”

12.粘贴刚刚复制的文件

13.选中【Tarch2014过期补丁】，鼠标右击选择“以管理员身份运行”

14.点击“补丁”

15.显示补丁完成，点击“退出”

16.双击“Tarch2014”

17.点击“是”

18.点击“确定”

19.双击图标，运行软件

20.勾选上下次不再提问，点击“确定”

21.取消勾选启动时显示，点击“关闭”

22.安装结束
