## · 软件介绍
C4D全名CINEMA 4D，由德国Maxon Computer研发出的3D动画软体。C4D是一个老牌的三维软件。能够进行顶级的建模、动画和渲染的3D工具包。内置纹理、动画、渲染、多边形建模、克隆、雕刻等多种辅助设计工具。
